# JavaScript Micro-Interactions Lab 🧪

A collection of vanilla JavaScript experiments to master DOM manipulation and state management. No frameworks, just code.

## Mission Statement

This repository is a **laboratory for mastering core JavaScript concepts** without the safety net of frameworks. Each project focuses on fundamental DOM interactions, event handling, and state logic—the building blocks that power modern web applications.

## How to Run

Getting started is simple:

1. Navigate to any project folder
2. Open the project's HTML file (for example `clickCounter.html` or `tipCalculator.html`) in your browser
3. Start interacting and experimenting!

No build tools, no dependencies—just pure JavaScript doing what it does best.
